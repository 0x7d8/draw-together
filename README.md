**Installation**

```sh
git clone https://github.com/0x7d8/draw-together.git
cd draw-together

# make sure nodejs 18+ is installed
npm i -g pnpm
pnpm install
pnpm build
```

<br/>

**Running**

```sh
cd draw-together

# use the --nosave argument to skip saving to disk
PORT=8000 pnpm prod
```
